适用于SolidWorks MBD零件的三维检索软件<br />
使用ATL和COM编程调用Solidworks API，利用OpenGL图形库实现三维模型的底层渲染与几何处理，基于ImGui开源框架构建图形用户界面。特征库的构建与深度学习算法的实现基于PyTorch深度学习框架。
